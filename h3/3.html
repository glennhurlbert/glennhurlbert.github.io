(a) Text #19.9.
First, suppose that T is not a tree. Then T contains a cycle C. Let v1, v2, ... vk be the nodes, in order, along C, let a1, a2, ... ak be the arcs, in order, along C (so that each ai joins ai to ai+1 in one direction or the other), and let c1, c2, ... ck be the columns of B coresponding to these arcs. [All subscript arithmetic is done modulo k.]

We will prove that B is singular by showing that these columns are linearly dependent. Define coeficients t1, t2, ... tk by ti=+1 if the arc ai goes from vi to vi+1, and ti=-1 if the arc ai goes from vi+1 to vi. We claim that the linear combination s = t1c1 + t2c2 + ... + tkck = 0.

We will prove this by showing that every entry of s is zero. Indeed, consider sj, which corresponds to the node vj. The only nonzero contributions to vj come from tj-1cj-1 and tjcj. Regardless of the direcion of the arc aj-1, the definition of tj-1 compensates so that the product tj-1cj-1=+1. Likewise, regardless of the direcion of the arc aj, the definition of tj compensates so that the product tjcj=-1. Hence, sj = tj-1cj-1 + tjcj = 0.

Second, suppose that T is a tree. We will show that B is nonsingular by proving that det(B) is nonzero. In fact, we will do this by permuting the rows and columns of B to create a matrix B' which is upper triangular and has no zeros on its diagonal. (Such permutations are elementary row and column operations which only multiply the determinant of B by +/-1. Hence Hence, |det(B)|=|det(B')|>0.)

We proceed by induction on n>=2. If n=2 then B=I1, the 1x1 identity matrix, so the result is true. If n>2 then let vn be a leaf of T (a node which is incident with only one arc) which is not the node corresponding to the truncated row of B (which is possible since there are always at least two leaves in any tree), and let an be the unique arc incident with vn. Let B' be so that its last row corresponds to vn, and so that its last column coresponds to an. Then the only nonzero entry of B' in the last row is in the last column.

Now let T* be the tree obtained from T by removing both vn and an. Then T* has n-1 nodes and n-2 arcs (and still no cycle, which is why it is a tree), and so by the induction hypothesis the rows and columns of its truncated incidence matrix can be permuted so that it is upper triangular with no zeros on its diagonal. Denote this matrix by B*, and let B' be identical to B* on its first n-1 rows and columns. Then B' is upper triangular with no zeros on its diagonal, as claimed.

This completes the proof.
