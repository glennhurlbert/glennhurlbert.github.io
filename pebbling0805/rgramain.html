Notation
The notion that graphs with very few edges tend to have large pebbling number and graphs with very many edges tend to have small pebbling number can be made precise as follows. Let Gn,p be the random graph model in which each of the {n choose 2} possible edges of a random graph having n vertices appears independently with probability p. For functions f and g on the natural numbers we write that f << g (or g >> f) when f/g --> 0 as n --> infinity. Let o(g) = {f | f << g} and define O(g) (resp., Omega(g)) to be the set of functions f for which there are constants c, N such that f(n) <= cg(n) (resp., f(n) >= cg(n)) whenever n > N. Finally, let Theta(g) = O(g) intersect Omega(g).

Thresholds
Let P be a property of graphs and consider the probability Pr[P] =Prp[P] that the random graph Gn,p has P. For large p it may be that Pr[P] --> 1 as n --> infinity, and for small p it may be that Pr[P] --> 0 as n --> infinity, More precisely, define the threshold of P, t(P), to be the set of functions t for which p >> t implies that Pr[P] --> 0 as n --> infinity.
Class 0 Threshold
It is not clear that such thresholds exist for arbitrary P. However, we observe that Class 0 is a monotone property (adding edges to a Class 0 graph maintains the property), and a theorem of Bollobas and Thomason [BoTh] states that t(P) is nonempty for every monotone P It is well known [ErRe] that t(connected) = Theta(logn/n), and since connectedness is required for Class 0, we see that t(Class 0) subset Omega(logn/n). In [CHH] it is noted that t(Class 0) subset O(1). In [CHKT] the Diameter-Connectivity Theorem was used to prove the following result.
Class 0 Threshold Theorem.   For all d > 0, t(Class 0) subset O((nlogn)1/d/n).
