Cells (tableau entries)
  • Pivot on Entry - performs an integer pivot on the cell; this entry becomes the basic coefficient for the resulting tableau.
  • Highlight Entry - changes the background color of the cell for emphasis; useful for classroom discussions (highlight all basic coefficients, potential pivot entries, etc.).
  • Add Row and Column After - adds a new row and new column after this cell; useful for adding new constraints (such as cutting planes, for example).
  • Delete Row and Column - deletes this row and column.
  • Make Entry 0 - enters 0 in cell (actually easier to simply type 0).
  • Make Entry 1 - enters 1 in cell (actually easier to simply type 1).
