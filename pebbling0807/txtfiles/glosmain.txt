<blockquote>
<a class="pdeflink" href="pnummain.html#2con">2-Connected</a>
<blockquote>
the property that a graph cannot be disconnected by removing only one vertex
</blockquote>
</blockquote>



<blockquote>
<a class="pdeflink" href="gconmain.html#2PP">2-Pebbling Property</a>
<blockquote>
two pebbles can be moved to any specified vertex when the initial
<a class="gdef" href="glosmain.html#config">configuration</a>
$C$ has size $2\pi(G)-s(C)+1$, where $s(C)$ is the size of the
<a class="gdef" href="glosmain.html#support">support</a>
</blockquote>
</blockquote>

<a name="A">

<a name="B">

<blockquote>
<a name="bip" class="pdeflink" href="pnummain.html#star">Bipartite Graph</a>
<blockquote>
a graph whose vertices can be covered by two sets of pairwise nonadjacent vertices
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#book">Book</a>
<blockquote>
the graph 
$S_{p+1} \times P_q$,
having
$p$
pages,
$q$
vertices
per page, and
$q$
vertices on the binding. 
</blockquote>
</blockquote>

<a name="C">

<blockquote>
<a class="pdeflink" href="gconmain.html#CaPr">Cartesian Product</a>
<blockquote>
<a class="mit">G<sub>1</sub></a> x G<sub>2</sub></a>
has vertex set
$V(G_1 \times G_2) = \{(v_1,v_2) | v_1 \in V(G_1), v_2 \in V(G_2) \}$
and edge set
$E(G_1 \times G_2) = \{\{(v_1,v_2),(w_1,w_2)\} | (v_1=w_1$ $and$ $(v_2,w_2) \in E(G_2))$ $or$
$(v_2=w_2$ $and$ $(v_1,w_1) \in E(G_1))\}$
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="clasmain.html#def">Class 0</a>
<blockquote>
the property that a graph has
<a class="gdef" href="glosmain.html#pebbnum">pebbling number</a>
equal to its number of vertices 
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#complete">Complete</a>
<blockquote>
the property that a graph has every vertex adjacent to all other vertices
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#star">Complete Bipartite Graph</a>
<blockquote>
a
<a class="gdef" href="glosmain.html#bip">bipartite graph</a>
in which every pair of vertices not belonging to the same partite set is adjacent
</blockquote>
</blockquote>


<blockquote>
<a name="config" class="pdeflink" href="pnummain.html#configuration">Configuration</a>
<blockquote>
a function that indicates the number of pebbles per vertex
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="clasmain.html#con">Connectivity</a>
<blockquote>
the minimum number of vertices whose deletion disconnects the graph or reduces it to one vertex
</blockquote>
</blockquote>

<blockquote>
<a name="cpn" class="pdeflink" href="cpebmain.html#pebnum">Cover Pebbling Number</a>
<blockquote>
the minimum number of pebbles needed so that from any initial
<a class="gdef" href="glosmain.html#config">configuration</a>
of the pebbles, after a series of pebbling moves, it is possible to have at 
least $1$ pebble on every vertex of a graph
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="cpebmain.html#cov">Cover Pebbling Ratio</a>
<blockquote>
for a graph
$G$,
the
<a class="gdef" href="glosmain.html#cpn">cover pebbling number</a>
of
$G$
divided by the
<a class="gdef" href="glosmain.html#pebbnum">pebbling number</a>
of
$G$
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="histmain.html#cross">Cross Number</a>
<blockquote>
for a sequence $S$ in a group, the value: 
$\displaystyle{\sum_{g \in S}\frac{1}{|g|}}$
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#cube">Cube</a>
<blockquote>
a graph that has all binary $d$-tuples as vertices and edges between that differ in exactly one coordinate
</blockquote>
</blockquote>

<blockquote>
<a name="cycle" class="pdeflink" href="pnummain.html#cycle">Cycle</a>
<blockquote>
a simple graph whose vertices can be placed on a circle so that vertices are adjacent if and only if they appear consecutively on the circle
</blockquote>
</blockquote>

<a name="D">
<blockquote>
<a class="pdeflink" href="histmain.html#dav">Davenport's Constant</a>
<blockquote>
the smallest $D$ such that every sequence of $D$ elements contains a
<a class="gdef" href="glosmain.html#zero">zero-sum subsequence</a>
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#diameter">Diameter</a>
<blockquote>
the maximum distance between two vertices of a graph 
</blockquote>
</blockquote>

<blockquote>
<a name="domset" class="pdeflink" href="pnummain.html#dominating">Dominating Set</a>
<blockquote>
a subset $S$ of the vertex set of a graph such that every vertex outside of $S$ has a neighbor in $S$ 
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#dominating">Domination Number</a>
<blockquote>
the minimum size of a
<a class="gdef" href="glosmain.html#domset">dominating set</a>
of vertices
</blockquote>
</blockquote>

<a name="E">

<a name="F">

<blockquote>
<a class="pdeflink" href="algomain.html#fra">Fractional Pebbling Number</a>
<blockquote>
$\liminf_{k \rightarrow \infty} \pi(G)/k$
</blockquote>
</blockquote>


<blockquote>
<a class="pdeflink" href="cpebmain.html#fuse">Fuse</a>
<blockquote>
the graph on
$n$
vertices composed of a path on
$l$
vertices with
$n-l$
independent vertices incident to one of its endpoints
</blockquote>
</blockquote>

<a name="G">
<blockquote>
<a class="pdeflink" href="clasmain.html#gir">Girth</a>
<blockquote>
the length of a shortest
<a class="gdef" href="glosmain.html#cycle">cycle</a>
in a graph
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="algomain.html#gre">Greedy Graph</a>
<blockquote>
a graph
$G$
for which every
<a class="gdef" href="glosmain.html#config">configuration</a>
of size
<a class="gdef" href="glosmain.html#pebbnum">$\pi(G)$</a>
has a
<a class="gdef" href="glosmain.html#greedy_soln">greedy slolution</a>
</blockquote>
</blockquote>

<blockquote>
<a name="greedy_soln" class="pdeflink" href="algomain.html#gre">Greedy Solution</a>
<blockquote>
a pebbling solution in which every step is
<a class="gdef" href="glosmain.html#greedy_step">greedy</a>
</blockquote>
</blockquote>

<blockquote>
<a name="greedy_step" class="pdeflink" href="gconmain.html#greedy">Greedy Step</a>
<blockquote>
a pebbling step from a vertex
$u$
to a vertex
$v$
such that
${\rm dist}(v,r)<{\rm dist}(u,r)$,
where
$r$
is the
<a class="gdef" href="glosmain.html#root">root</a>
vertex
</blockquote>
</blockquote>

<a name="H">
<blockquote>
<a class="pdeflink" href="histmain.html#hsum">H-Sum Sequence</a>
<blockquote>
for a subgroup $H$ of a group $G$, a sequence of elements of $G$ that sums to an element of $H$ 
</blockquote>
</blockquote>

<a name="J">

<a name="K">
<blockquote>
<a class="pdeflink" href="clasmain.html#kne">Kneser Graph</a>
<blockquote>
the graph with
vertices
${[m] \choose t}$
and edges
$\{A,B\}$
whenever
$A$
and
$B$
are disjoint
</blockquote>
</blockquote>

<a name="L">

<a name="M">

<blockquote>
<a name="majorizes" class="pdeflink" href="pnummain.html#maj">Majorize</a>
<blockquote>
the sequence
$(s_1,\ldots,s_k)$
majorizes the sequence
$(t_1,\ldots,t_k)$
if
$s_i > t_i$
when
$s_j=t_j$
for all
$j<i$
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#mpp">Maximum Path Partition</a>
<blockquote>
a
<a class="gdef" href="glosmain.html#pp">path partition</a>
of a
<a class="gdef" href="glosmain.html#tree">tree</a>
$T$
such that no other path partition
<a class="gdef" href="glosmain.html#majorizes">majorizes</a>
it
</blockquote>
</blockquote>

<a name="N">

<a name="O">
<blockquote>
<a class="pdeflink" href="opebmain.html#optimal">Optimal Pebbling Number</a>
<blockquote>
the size of the smallest solvable
<a class="gdef" href="glosmain.html#config">configuration</a>
in a graph
</blockquote>
</blockquote>

<a name="P">
<blockquote>
<a name="path" class="pdeflink" href="pnummain.html#path">Path</a>
<blockquote>
a sequence of vertices, each adjacent to its successor 
</blockquote>
</blockquote>

<blockquote>
<a name="pp" class="pdeflink" href="pnummain.html#wrd">Path Partition</a>
<blockquote>
an
<a class="gdef" href="glosmain.html#rpp">$r$-path partition</a>
for some
$r$
</blockquote>
</blockquote>

<blockquote>
<a name="pebbnum" class="pdeflink" href="pnummain.html#pebnum">Pebbling Number</a>
<blockquote>
the fewest number of pebbles that gurantee that a specific graph is
<a class="gdef" href="glosmain.html#solvable">solvable</a> 
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="rpebmain.html#thre">Pebbling Threshold</a>
<blockquote>
a
<a class="gdef" href="glosmain.html#thresh">threshold</a>
for the
<a class="gdef" href="glosmain.html#solvable">solvability</a>
of random pebbling
<a class="gdef" href="glosmain.html#config">configurations</a>
on graph sequences
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#petersen">Petersen Graph</a>
<blockquote>
the disjointness graph of the $2$-sets in a $5$-element set
</blockquote>
</blockquote>
 
<blockquote>
<a class="pdeflink" href="cpebmain.html#pebnum">Positive Weight Function</a>
<blockquote>
$w(v)>0$
for every vertex
$v$,
where
$w(v)$
is the number of pebbles on vertex
$v$
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#maj">$r$-Maximum Path Partition</a>

<a class="gdef" href="glosmain.html#rpp">$r$-path partition</a>

<blockquote>
<a name="rpp" class="pdeflink" href="pnummain.html#wrd">$r$-Path Partition</a>

$Q=(Q_1,\ldots,Q_m)$
of the edges of a
<a class="gdef" href="glosmain.html#tree">tree</a>
$T$

<a class="gdef" href="glosmain.html#path">paths</a>

$Q_i$

<a class="gdef" href="glosmain.html#wrd">well $r$-directed</a>

<blockquote>
<a name="rsol" class="pdeflink" href="pnummain.html#rsol">r-Solvable</a>
<blockquote>
the property that there is a sequence of pebbling moves
for a given graph such that it is possible to place a pebble on the
<a class="gdef" href="glosmain.html#root">root</a> 
</blockquote>
</blockquote>

<blockquote>
<a class="pdeflink" href="pnummain.html#rsol">r-Unsolvable</a>
<blockquote>
the property that for a given graph, it is not possible to place a pebble on the
<a class="gdef" href="glosmain.html#root">root</a> 
through a sequence of pebbling moves 
</blockquote>
</blockquote>

<blockquote>
<a name="root" class="pdeflink" href="pnummain.html#root">Root</a>

</blockquote>
</blockquote>

<a name="S">
<blockquote>
<a class="pdeflink" href="gconmain.html#greedy">Semi-greedy</a>
<blockquote>

to a vertex

<blockquote>
<a name="solvable" class="pdeflink" href="pnummain.html#rsol">Solvable</a>

<a name="T">
<blockquote>
<a name="thresh" class="pdeflink" href="rgramain.html#thresh">Threshold</a>
<blockquote>

$\mathbf{P}$
of a random structure
$S_p$,
the set of functions
$t$
for which
$p \gg t$
implies that
$Pr[S_p$ $has$ $\mathbf{P}] \rightarrow 0$
as
$n \rightarrow \infty$,
and
$p \ll t$
implies that
$Pr[S_p$ $has$ $\mathbf{P}] \rightarrow 1$
as
$n \rightarrow \infty$
</blockquote>
</blockquote>
